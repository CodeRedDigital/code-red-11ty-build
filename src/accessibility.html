Why #a11y for short? a11y is a numeronym of accessibility, meaning it starts with an 'A' has 11 letters and ends with a 'Y'.

Web Development

HTML, or HyperText Markup Language, is the foundations of the web. HTML was created by Sir Tim Berner-Lee as a way of connecting documents together and since that day, back in 1990, this has been the way that websites and online services work.

HTML is an incredibly smart language and has been built to be backwards compatible, what this means is that any site, service, portal will work on any browser in the world, even those built by Tim in 1990.

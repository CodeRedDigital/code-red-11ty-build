This accessibility statement applies to code-red.uk

This website is run by Code Red. It is designed to be used by as many people as possible. The text should be clear and simple to understand. You should be able to:

  • zoom in up to 400% without problems
  • navigate most of the website using just a keyboard
  • navigate most of the website using speech recognition software
  • use most of the website using a screen reader (including the most recent versions of JAWS, NVDA and VoiceOver)

Feedback and contact information

While every effort has been made to make sure that the content on this site is accessible, if you have a problem with any page(s), please let us know and we shall rectify the issue. When contacting us please porovide the following information:

  • the web address (URL) of the content
  • your email address and name
  • what issue you are experiencing with the content
